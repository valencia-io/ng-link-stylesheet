This view has and style3.css stylesheet with the attribute ng-link-stylesheet-no-destroy

If the attribute ng-link-stylesheet-no-destroy is found, the <link href="style.css"> inserted won't be removed when the scope is destroyed, which is the normal behavior. So the style3.css will be in the head until the page is refreshed. In this case: 

      body {
          background-color: #ffcc00!important;
      }
